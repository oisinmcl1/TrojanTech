Designed for Palo Alto Firewalls

The script retrieves the default gateway as the IP which assuming it's the firewall.

Get-APIKey
Uses admin as default username.
Takes the default gateway and password that you input and retrieves API key.

Export-FirewallConfig
Retrieves firewall config as xml file using API key to your desired directory.
